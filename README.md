# Image-to-Json_LLM
Finetune LLM to convert an invoice or receipt image to receipt JSON object.

```
# !pip install -q -U einops
# !pip install -q -U safetensors
# !pip install -q -U torch
# !pip install -q -U xformers
# !pip install scipy
```
